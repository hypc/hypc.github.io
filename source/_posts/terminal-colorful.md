**控制代码**

```
\033[0m             关闭所有属性
\033[1m             设置高亮
\033[4m             下划线
\033[5m             闪烁
\033[7m             反显
\033[8m             消隐
\033[30m--\033[37m  设置前景色：黑、深红、绿、黄、蓝、紫、深绿、白
\033[40m--\033[47m  设置背景色：黑、红、绿、黄、蓝、紫、深绿、白
\033[nA             光标上移n行
\033[nB             光标下移n行
\033[nC             光标右移n列
\033[nD             光标左移n列
\033[y;xH           设置光标的位置
\033[2J             清屏
\033[K              清除从光标列到行尾的内容
\033[s              保存光标位置
\033[u              恢复光标位置
\033[?25l           隐藏光标
\033[?25h           显示光标
```

**例**

```bash
echo -e '\033[31mcontent\033[37m'
```
